# Fluorite

![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/FluoriteOS/Fluorite/build.yml?style=for-the-badge&logo=github&label=Container%20Build)

**Fluorite** is a *uBlue* image built for the power user.

It uses *Niri* as a desktop, and comes included with a bunch of handy tools, including:

- Ghostty
- Starship (must be activated manually in your bashrc)
- Warehouse
- Flatseal
- Ruby
- Tmux
- Zen Browser
- and more!

## Documentation

There is currently no documentation, but that could change in future!
